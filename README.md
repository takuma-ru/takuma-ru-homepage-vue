## Getting Started
#### 1. Install
```powershell
yarn
```

#### 2. Start dev server
```powershell
yarn dev
```

### 3. Start Storybook server
```powershell
yarn storybook
```

#### 4. Build this App
```powershell
yarn generate
```

```powershell
yarn preview
```

### Hygen commands
- Create Component
  ```powershell
  yarn hygen:component
  ```
- Create Composable
  ```powershell
  yarn hygen:composable
  ```
- Create Page
  ```powershell
  yarn hygen:page
  ```
- Create Util script
  ```powershell
  yarn hygen:util
  ```
